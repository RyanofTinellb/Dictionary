xi

High Lulani

(X)

/ɕiː/

transitive stem verb
to appear to be; to project
Thematic Relations

stimulus subject

experiencer dative

theme object

Example

ʔusu ba’u ximussa ra’u?

(uz / bu / XU;s / ru?)

1int 2dat project-exoticism prs;sta?

“Do I seem exotic to you?”

Notes

used for subtle themes. Cf: xu (Z) broadcast.

Derived Terms

xi’axasi (XaxS) to be mature

xipiki (XPK) to be clear

xijapu (Xjo) to be sincere

ximussa (XU;s) to be exotic

xiranna (Xr;n) to be exalted

xifipu (XFo) to be rude

xihuba (XVb) to be important
